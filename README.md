# solar-scale

A small tool for making the solar system feel real.

Give it a scale and it tells you where everything goes.

---

## why this exists

It started with a question:

What if Neptune sat 142 feet from the sun?

That turned into measurements.  
Then into layouts across a yard.  
Then into something worth keeping.

This is that something.

---

## what it does

- scales planetary distances from a chosen reference point
- outputs positions in feet and sizes in inches
- built as both a CLI and an API

Nothing fancy. Just useful.

---

## quick start

```bash
uv run solar-scale calculate --neptune-distance-ft 142
